# Концептуальная модель предметной области

Наша задача -- автоматизировать документооборот для защиты научной диссертации.

В защите любой диссертации принимают участие следующие **субъекты**:

1. Диссертационный совет: его Председатель и Секретарь
2. сам Соискатель
3. его Научный руководитель
4. Оппоненты (не менее 2, но не более `скольких?`)

В защите любой диссертации фигурируют следующие **объекты**:

1. Диссертация: автореферат, текст итд
2. различные ачивки: документы об образовании, публикации по теме диссертации
3. Протокол о создании комиссии
4. Протокол о приеме к защите
5. Заключение организации
6. Заключение комиссии ДС по предварительному рассмотрению
7. Скриншот опубликования на сайте МИФИ
8. Паспорт специальности
9. Поянительная записка для подписи заключения
